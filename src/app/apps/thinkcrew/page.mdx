## Solution

<AppSolution image='thinkcrew_diagram.png' href={appData.url}>
  
Thinkcrew is split in 3 different systems. There are Projects, Scripts and Schedule parts.

The first part Projects is a project management part and users can control all project in this part.
In project, is includes all scripts and schedules for every project.

The second part Scripts is a script management part and users can make, edit, and managing elements
In script, users can set schedule for every person and element

The third part Schedule is a schedule management part and users can make, update schedul and analizing result.
In Schedule, users can manage all schedule for persons, elements and analize all project result.
